Gradient Descent on $L(\theta) = \tfrac{1}{2} S\,\theta^2$

Both panels use the same learning rate $\eta$ but different curvatures: the left has $S = 1.3$ and the right has $S = 2.0$. The update rule is $\theta_{t+1} = (1 - \eta S)\,\theta_t$, so convergence requires $\eta < 2/S$. With $S=1.3$ the threshold is $\eta < 1.54$; with $S=2$ it is $\eta < 1.0$. At the default $\eta = 1.2$ the left panel converges and the right diverges. Drag $\theta_0$ or adjust $\eta$ to explore both regimes.

Gradient Descent on $L(\mathbf{w}) = \tfrac{1}{2}(\lambda_1 \tilde{w}_1^2 + \lambda_2 \tilde{w}_2^2)$

A quadratic loss in 2-D with independent curvatures $\lambda_1$ and $\lambda_2$ along its eigenvectors $q_1$ and $q_2$. GD converges only if $\eta < 2/\lambda_i$ in every direction — one unstable direction is enough to diverge. Drag the red dot to set a new starting point.

Gradient Descent on $L(x,y) = (1 - ax^2 - y^2)^2$

Minimum manifold: ellipse $ax^2 + y^2 = 1$. When $a = 1$ this is a circle; $a \neq 1$ stretches or compresses it along $x$. The loss landscape uses a log scale colourmap (blue = low, red = high). The sharpness landscape shows $\lambda_{\max}$ of the Hessian on a linear scale with white pinned at $2/\eta$ — the critical threshold — and a black contour line at that value. Drag the white dot to choose a starting point $\theta_0 = (x_0, y_0)$.
